# Korea AI Trader V1.1 Backtest Lab

## 핵심

V1.0 전략을 그대로 유지하면서, 웹 대시보드에서 직접 백테스트 파라미터를 조정하고 실행할 수 있게 만든 버전입니다.

## 추가 기능

- 백테스트 설정 폼
- 최소 AI 점수 조정
- 이벤트당 진입 종목 수 조정
- 진입 방식: 다음날 시가 / 당일 종가
- 1차 익절, 1차 익절 비중, 최종 익절, 손절, 최대 보유일 조정
- MA20 이탈 청산 ON/OFF
- 매수 사이드카 탈출 ON/OFF
- 추세필터 ON/OFF
- 누적 손익 곡선
- 월별 성과
- 거래별 진입 근거/청산 이유

## 배포

1. GitHub 기존 파일 삭제
2. zip 압축 풀고 안의 파일들을 루트에 업로드
3. Railway 자동 배포
4. 메인 URL 접속
